# Audio Files

Letakkan file audio MP3 di folder ini:

- `track1.mp3` → Wonderful Indonesia (track pertama)
- `track2.mp3` → Pesona Nusantara (track kedua)

File audio akan otomatis diputar ketika halaman dibuka.

> **Catatan:** Gunakan file MP3 dengan ukuran yang tidak terlalu besar agar loading cepat.
> Contoh sumber musik bebas royalti: https://pixabay.com/music/ atau https://freesound.org/
